Description

A fully functional 2-player Connect Four game built using basic HTML, CSS, and JavaScript.
The game runs entirely in the browser and allows two players to take turns dropping pieces into a 7-column grid.

Features

2-player local gameplay

Turn-based system

Win detection (horizontal, vertical, diagonal)

Draw detection

Game reset functionality

Clean and responsive UI

Technologies Used

HTML5

CSS3

JavaScript

Skills Demonstrated

DOM manipulation

Event handling

Game logic implementation

Conditional logic & algorithms

State management

Basic UI styling

How to Run

Download or clone the repository

Open index.html in your browser

Start playing 🎮
